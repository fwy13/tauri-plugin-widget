## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-set-items`
- `allow-get-items`
- `allow-set-register-widget`
- `allow-reload-all-time-lines`
- `allow-reload-time-lines`
- `allow-request-widget`
